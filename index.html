隐私权政策 / Privacy Policy

Spore - Knowledge Absorber & Memory Engine

最后更新 / Last Updated：2026 年 5 月 23 日

核心原则：本地优先，用户掌控。
Spore 的所有核心数据均存储在您的浏览器本地，我们没有任何服务器来收集或存储您的个人信息。您配置的 API Key 仅存于本地，仅用于向您指定的 AI 服务商发送请求。

Core Principle: Local-First, User-Controlled.
All of Spore's core data is stored locally in your browser. We operate no servers that collect or store your personal information. Your API keys are stored locally and used solely to send requests to the AI provider you configure.

1. 数据收集与存储 Data Collection & Storage

中Spore（以下简称"本扩展"）采用本地优先（Local-First）架构：

  • 文章与笔记数据：您从网页捕获的文章、AI 生成的摘要、闪卡（记忆卡片）、笔记等所有知识数据，均存储在浏览器本地的 IndexedDB 中。
  • API Key 配置：您自行配置的 AI 服务商 API Key（如 OpenRouter、DeepSeek、SiliconFlow 等），仅存储在浏览器 chrome.storage.local 中。
  • 使用偏好：您的分类体系、复习进度、主题偏好等设置数据，均存储在浏览器本地。

本扩展没有任何云端服务器。我们不会收集、存储或转发您的任何数据。

ENSpore adopts a Local-First architecture:

  • Articles & Notes: All knowledge data—captured articles, AI-generated summaries, flashcards, notes, etc.—is stored in your browser's local IndexedDB.
  • API Key Config: API keys you configure for AI providers (e.g., OpenRouter, DeepSeek, SiliconFlow) are stored only in chrome.storage.local.
  • Preferences: Your taxonomy, review progress, theme preferences, and other settings are stored locally in your browser.

This extension operates no cloud servers. We do not collect, store, or relay any of your data.

2. 数据传输 Data Transmission

中本扩展唯一对外发起网络请求的场景是：

  • AI 内容处理：当您使用 AI 总结、闪卡生成、创意工坊等功能时，文章内容会被发送到您在设置页面中自行配置并授权的 AI 服务商 API（如 OpenRouter、DeepSeek、SiliconFlow 等）。

除此之外，本扩展不会向任何第三方服务器传输数据。所有 API 通信均通过 HTTPS 加密进行。我们不包含任何分析 SDK、广告 SDK 或第三方追踪代码。

ENThe only outbound network requests made by this extension are:

  • AI Content Processing: When you use features like AI summarization, flashcard generation, or Forge Studio, article content is sent to the AI provider API you have configured and authorized (e.g., OpenRouter, DeepSeek, SiliconFlow).

Beyond this, the extension does not transmit data to any third-party servers. All API communications are encrypted via HTTPS. We do not include any analytics SDK, advertising SDK, or third-party tracking code.

3. 信息使用 Data Usage

中本扩展在本地处理您的数据，仅用于以下目的：

  • 在侧边栏中展示和管理您的知识库
  • 调用您配置的 AI API 生成摘要、闪卡和多格式内容
  • 通过本地 SRS（间隔重复）算法安排复习提醒
  • 生成本地数据洞察看板（阅读统计、领域分布等）

ENAll data is processed locally and used solely for:

  • Displaying and managing your knowledge base in the side panel
  • Calling your configured AI API to generate summaries, flashcards, and multi-format content
  • Scheduling review reminders via the local SRS (Spaced Repetition System) algorithm
  • Generating local data insight dashboards (reading stats, domain distribution, etc.)

4. 信息共享 Information Sharing

中本扩展不会向任何第三方出售、出租或分享您的个人信息。

您配置的 AI 服务商在处理 API 请求时，将按照其自身的隐私政策处理数据。建议您在使用前查阅所选 AI 服务商的隐私政策。

ENThis extension does not sell, rent, or share your personal information with any third party.

The AI provider you configure will process data according to its own privacy policy when handling API requests. We recommend reviewing the privacy policy of your chosen AI provider before use.

5. 权限说明 Permissions

中本扩展申请了以下权限及其用途：

  • storage：在浏览器本地存储文章、闪卡、设置等所有数据
  • sidePanel：在侧边栏中展示知识库面板
  • tabs / activeTab：获取当前标签页信息以便捕获文章内容
  • contextMenus：提供右键菜单快速保存网页文章
  • alarms：定时触发 SRS 复习提醒
  • notifications：发送复习提醒通知
  • host_permissions (<all_urls>)：在用户主动操作时，从当前网页提取正文内容

ENThis extension requests the following permissions and their purposes:

  • storage: Store articles, flashcards, settings, and all other data locally in your browser
  • sidePanel: Display the knowledge base panel in the browser side panel
  • tabs / activeTab: Access current tab information to capture article content
  • contextMenus: Provide right-click context menu for quick article saving
  • alarms: Schedule SRS review reminders
  • notifications: Send review reminder notifications
  • host_permissions (<all_urls>): Extract main content from the current webpage when you actively use the extension

6. 数据删除 Data Deletion

中您可以随时通过以下方式删除所有数据：

  • 在扩展侧边栏或仪表盘中手动删除文章或闪卡
  • 在浏览器扩展管理页面中清除 Spore 的本地存储数据
  • 卸载本扩展将自动清除所有存储的本地数据

ENYou can delete all data at any time by:

  • Manually deleting articles or flashcards in the side panel or dashboard
  • Clearing Spore's local storage data in your browser's extension management page
  • Uninstalling this extension, which will automatically remove all stored local data
