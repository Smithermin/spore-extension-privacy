隐私权政策

最后更新：2026 年 5 月 23 日

DeepSeek Mate（以下简称"本扩展"）尊重并保护您的隐私。本隐私权政策说明本扩展如何收集、使用和保护您的信息。

1. 信息收集

本扩展在您使用过程中可能会收集以下信息：

  • 对话内容：当您使用收藏功能时，本扩展会保存您当前对话中的问题与回答内容。这部分数据仅存储在您的浏览器本地（chrome.storage.local），不会上传至任何第三方服务器。
  • 历史对话记录：当您启用历史同步功能时，本扩展会通过 DeepSeek 官方 API 获取您在 chat.deepseek.com 上的历史对话记录。这些数据同样仅存储在您的浏览器本地。
  • 提示词数据：您自定义的提示词内容将存储在浏览器本地。

2. 信息使用

本扩展收集的信息仅用于以下目的：

  • 在侧边栏中展示您收藏的对话和历史记录
  • 提供本地搜索功能，帮助您快速查找历史对话
  • 导出对话为 Markdown 格式
  • 创建分享链接（通过 DeepSeek 官方 API）

3. 信息共享

本扩展不会向任何第三方出售、出租或分享您的个人信息。本扩展不包含任何分析、广告或追踪代码。

本扩展与 DeepSeek 官方 API 的通信仅用于：

  • 获取您授权访问的历史对话记录
  • 创建对话分享链接

这些通信基于您在 DeepSeek 网站上的登录状态和授权。

4. 数据存储

所有数据均存储在您的浏览器本地存储（chrome.storage.local）中。您可以随时通过以下方式管理或删除数据：

  • 在扩展侧边栏中手动删除收藏的对话
  • 在 Chrome 扩展管理页面中清除扩展的本地存储数据
  • 卸载本扩展将自动清除所有存储的数据

5. 数据传输

本扩展不会将您的数据传输至任何除 DeepSeek 官方服务器之外的第三方服务器。所有与 DeepSeek API 的通信均通过 HTTPS 加密进行。

6. 权限说明

本扩展申请了以下 Chrome 权限及其用途：

  • storage：在浏览器本地存储收藏的对话、历史记录和设置
  • sidePanel：在侧边栏中展示收藏和搜索功能
  • scripting：在 chat.deepseek.com 页面注入收藏按钮等功能
  • notifications：收藏成功时发送通知提醒
  • downloads：导出对话为 Markdown 文件
  • tabs：获取当前标签页信息以管理对话
  • host_permissions (chat.deepseek.com)：仅在 DeepSeek 网站上运行

7. 儿童隐私

本扩展不面向 13 岁以下儿童，不会故意收集儿童的个人信息。

8. 隐私权政策的变更

本隐私权政策可能会不时更新。如有重大变更，我们会在扩展的更新说明中通知您。
